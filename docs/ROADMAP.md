# Roadmap

## V1：基础知识库问答

- 导入网站内容到 Dify Knowledge。
- 创建基础问答助手。
- 基于知识库回答用户问题。
- 无答案时拒答。

状态：已完成。

---

## V1.5：知识库优先 + sitemap 实时补充

- 知识库不足时读取 sitemap。
- 从 sitemap 中选择相关文章 URL。
- 使用 Jina Reader 实时读取文章正文。
- 基于实时文章回答。
- 无答案拒答。

状态：已完成原型。

---

## V2：自动同步知识库

计划能力：

- GitHub Actions 定时读取 sitemap。
- 判断新增/修改文章。
- 调用 Jina Reader 抓取 Markdown。
- 调用 Dify Knowledge API 更新知识库。
- 生成同步日志。

目标：减少手动维护成本。

---

## V2.1：评估与反馈系统

计划能力：

- 用户反馈按钮。
- badcase 自动收集。
- 测试集扩展到 50 条。
- 按类型统计失败原因。

---

## V3：多站点知识问答

计划能力：

- 支持多个 sitemap。
- 支持多知识库路由。
- 支持根据问题自动选择知识源。
- 支持后台统计高频问题和知识缺口。
